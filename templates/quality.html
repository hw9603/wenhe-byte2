Quality: About This Data Set

The quality of this data set is overall good, except just a few blank values and a little noise. Apart from that, the data set is clean and well structured.

Is the data complete?

The original data is not very complete. It has 4 out of 100 rows missing author's name. It raises the concern that it is hard to find out the top ten profitable author since 4 is not a small number comparing to 100.
We can see this from the visualization of the pie chart describing relation between the author and the volumes of books written by him. The graph is shown below with the top ten results included. The rating of N/As is 7, which avoids people to know the true results.

Is the data coherent?

It quite makes sense relative to itself and matches the common sense. We can see from the above pie chart that J.K. Rowling wrote books with the most number of volumes followed by Dan Brown, which quite makes sense.
If we look at another pie chart that illustrates the top ten books with the largest volume, we can see that The Da Vinci Code ranks the first, followed by books of Harry Potter series.
Another angle to view it is to see the performance in each categories of book, which is shown below. We can see that children's fiction ranks first, followed by crime thriller and adventure. This agrees with the common sense.

Is the data correct?

I can verify from the aspect that the more volume, the larger value. By doing so, I plot a chart with value and volume changing with respect to the publication date, as is shown below. We can find the trend that matches with our assumption. Also, there is a trend of increasing value as time goes by, which is reasonable according to our common sense.
I can also verify the value and volume according to its binding. In the below chart, I includes relationship between the two ways of binding and the value and volume. We can see that even though the volume of paperback is higher than that of hardback, the value of the hardback is greater than that of paperback. This is reasonable since hardback is usually more expensive than the paperback.
